# energy-toolkit-site

Source for [toolkit.energy](https://toolkit.energy) — the Energy Toolkit website.

Static SvelteKit site built with `@sveltejs/adapter-static`.

## Develop

```bash
npm install
npm run dev
```

Dev server runs on port 5174.

## Build

```bash
npm run build
```

Output lands in `build/`, ready for any static host.
